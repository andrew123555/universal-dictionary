# Gemini Yahoo 字典助手 📖

在 Gemini 網頁版快速查詢 Yahoo 字典並播放發音！

## ✨ 功能特色

- 🎯 在 Gemini 翻譯結果中選取單字，自動顯示「查字典」按鈕
- 📖 即時查詢 Yahoo 字典，顯示音標和釋義
- 🔊 播放單字發音（支援多個音訊來源和備用 TTS）
- 🎨 精美的介面設計，不干擾 Gemini 使用體驗
- ⚡ 無需 API Key，完全免費使用

## 📦 安裝步驟

### 1. 下載擴充功能檔案

將以下檔案放在同一個資料夾中：
- `manifest.json` - 擴充功能配置檔
- `content.js` - 主要功能腳本
- `styles.css` - 樣式檔案
- `icon.png` - 圖示檔案

### 2. 安裝到 Chrome 瀏覽器

1. **開啟 Chrome 擴充功能頁面**
   - 在 Chrome 網址列輸入：`chrome://extensions/`
   - 或點選「更多工具」→「擴充功能」

2. **啟用開發人員模式**
   - 點選右上角的「開發人員模式」開關

3. **載入擴充功能**
   - 點選「載入未封裝項目」按鈕
   - 選擇存放上述檔案的資料夾
   - 點選「選擇資料夾」

4. **完成！**
   - 你應該會看到「Gemini Yahoo 字典助手」出現在擴充功能列表中
   - 圖示會顯示一個紫色漸層的書本圖案

## 🚀 使用方法

### 基本使用

1. **前往 Gemini 網頁版**
   - 打開 https://gemini.google.com

2. **進行翻譯**
   - 在 Gemini 中輸入要翻譯的文字
   - 等待 Gemini 顯示翻譯結果

3. **查詢單字**
   - 用滑鼠選取翻譯結果中的任何英文單字
   - 自動彈出「📖 查字典」按鈕
   - 點擊按鈕

4. **查看結果**
   - 彈出視窗顯示：
     * 單字本身
     * 音標（如果有）
     * 中文釋義（最多 5 個）
     * 查看完整內容的連結

5. **播放發音**
   - 點擊「🔊 發音」按鈕即可聽到單字發音
   - 如果第一個音訊來源失敗，會自動使用瀏覽器的 TTS 功能

### 進階技巧

- **快速關閉**：點擊彈窗右上角的 ✕ 按鈕
- **查看完整內容**：點擊彈窗底部的連結，在新分頁開啟 Yahoo 字典完整頁面
- **選取其他單字**：關閉彈窗後，可以繼續選取其他單字查詢

## 🐛 常見問題

### Q1: 為什麼選取單字後沒有出現按鈕？
**A:** 
- 確認選取的是「純英文單字」（不包含數字或符號）
- 重新整理 Gemini 頁面
- 檢查擴充功能是否已啟用

### Q2: 為什麼查詢失敗？
**A:**
- 確認網路連線正常
- Yahoo 字典可能暫時無法存取
- 嘗試重新查詢

### Q3: 發音無法播放？
**A:**
- 第一次播放時瀏覽器可能會詢問是否允許音訊
- 如果第一個來源失敗，會自動使用瀏覽器 TTS
- 確認瀏覽器音量設定

### Q4: 可以在其他網站使用嗎？
**A:**
- 目前僅支援 Gemini 網頁版
- 如需在其他網站使用，可修改 `manifest.json` 中的 `matches` 設定

### Q5: 如何更新擴充功能？
**A:**
1. 下載新版本的檔案
2. 覆蓋原有檔案
3. 在 `chrome://extensions/` 頁面點擊擴充功能的「重新載入」按鈕
